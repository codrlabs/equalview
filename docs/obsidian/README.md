# Obsidian Vault

This folder is an [Obsidian](https://obsidian.md) vault used for brainstorming, diagramming, and note-taking.

## Contents

- **`equalview.canvas`** - Obsidian canvas with visual diagrams and notes
- **`*.png`** - Images pasted into the canvas for reference

## Purpose

This is supplementary brainstorming material, not the primary documentation. The main documentation lives in the [`../guides/`](../guides/) folder.

## Editing

Open this folder as a vault in Obsidian to edit the canvas and diagrams. For the markdown documentation, see the [`../guides/`](../guides/) folder.
